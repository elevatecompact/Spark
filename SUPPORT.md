# Support

The Spark community and team are here to help. Choose the channel that fits your needs.

## Community Support

| Channel | Purpose | Location |
|---------|---------|----------|
| GitHub Discussions | Feature requests, Q&A, feedback | https://github.com/spark/spark/discussions |
| Discord | Real-time chat, community help | https://discord.gg/spark |
| Stack Overflow | Technical questions (tag `spark-platform`) | https://stackoverflow.com |

## Official Support

For platform issues, billing, or account support, contact the Spark team through the following channels:

- **Email**: support@sparkplatform.com
- **Status Page**: https://status.sparkplatform.com
- **Help Center**: https://help.sparkplatform.com

## Reporting Bugs

Open a GitHub issue using the bug report template. Include:
1. Steps to reproduce
2. Expected vs actual behavior
3. Environment details (browser, OS, app version)
4. Logs or screenshots if available

## Feature Requests

Open a GitHub Discussion in the Ideas category with a clear description of the problem you want to solve and how you envision the feature working.

## Enterprise Support

Enterprise customers have access to a dedicated support team with SLAs. Contact enterprise@sparkplatform.com for details.
