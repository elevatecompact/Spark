# Configuration

| Key | Default | Description |
|-----|---------|-------------|
| s3.endpoint | s3.amazonaws.com | S3 endpoint |
| s3.region | us-east-1 | AWS region |
| s3.bucket_prefix | titan- | Bucket name prefix |
| s3.max_upload_size | 5GB | Max upload size |
| storage.hot_tier_days | 30 | Days in hot tier |
| storage.warm_tier_days | 90 | Days in warm tier |
| storage.cold_tier | GLACIER | Cold storage class |
| thumbnail.width | 320 | Default thumbnail width |
| transform.max_dimension | 4096 | Max transform dimension |
