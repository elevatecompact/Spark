# Changelog

## 0.5.0 (2026-06-22)
- Multi-tier storage lifecycle.
- Direct-to-S3 presigned URL uploads.
- Virus scanning with ClamAV.
- S3 Transfer Acceleration support.

## 0.4.0 (2026-04-25)
- Image transformation with ImageMagick.
- Video thumbnail generation.
- Multipart upload for large files.
- Asset trash and restore.

## 0.3.0 (2026-02-20)
- Content-addressable storage.
- PostgreSQL metadata index with Redis cache.

## 0.2.0 (2026-01-10)
- Basic S3 upload/download with presigned URLs.
