# Oracle Engine

**Purpose:** ML-powered recommendation engine for personalized content discovery.
**Tech Stack:** Python, PyTorch, Redis, ClickHouse, ONNX, gRPC, Apache Kafka.

Oracle delivers real-time personalized recommendations across video, clip, and live-stream content. It combines collaborative filtering, content-based embeddings, and a real-time behavioural stream processor to generate ranked recommendation lists. Models are trained offline and served via ONNX runtime with online feature updates through Redis.
