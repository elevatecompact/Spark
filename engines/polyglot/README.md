# Polyglot Engine

**Purpose:** AI-powered multilingual translation engine for real-time content localisation.
**Tech Stack:** Python, PyTorch, Transformers, Opus-MT, NLLB, ONNX, Redis, gRPC, Kafka.

Polyglot translates user-generated content, metadata, and live captions across 100+ languages. Combines neural machine translation with language detection, terminology management, and quality estimation. Supports batch VOD translation and real-time streaming translation for live captions.
