# Changelog

## 1.3.0 (2026-06-14)
- Kafka-backed durable delivery.
- Channel-level rate limiting with token bucket.
- Connection draining for zero-downtime upgrades.
- OpenTelemetry tracing.

## 1.2.0 (2026-04-08)
- Redis Cluster for multi-region Pub/Sub.
- Presence tracking with user-status channels.
- Server-side channel ACL.

## 1.1.0 (2026-02-10)
- Binary message support.
- Backpressure with slow-consumer disconnection.

## 1.0.0 (2026-01-05)
- Initial release: WebSocket gateway, channel pub/sub.
