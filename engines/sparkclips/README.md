# Sparkclips Engine

**Purpose:** AI-powered highlight detection and clip generation for live and VOD content.
**Tech Stack:** Python, PyTorch, Whisper, CLIP, FFmpeg, Redis, gRPC, Kafka, S3.

Sparkclips identifies exciting moments in live streams and recorded video using multimodal AI - audio analysis, visual event classification, and chat sentiment. Generates highlight reels and short-form clips optimised for social media sharing.
