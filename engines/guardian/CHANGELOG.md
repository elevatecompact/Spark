# Changelog

## 1.3.0 (2026-06-17)
- WebAuthn passkey support.
- Token binding with holder-of-key JWTs.
- Adaptive rate limiting with exponential backoff.
- Breached password detection.

## 1.2.0 (2026-04-15)
- OIDC compliance with userinfo endpoint.
- RBAC engine with resource-level permissions.
- Session management with concurrent limits.

## 1.1.0 (2026-02-20)
- OAuth 2.0 authorization code flow with PKCE.
- API key management.

## 1.0.0 (2026-01-10)
- Initial release: JWT issuance, password auth.
