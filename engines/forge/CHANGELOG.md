# Changelog

## 0.4.0 (2026-06-18)
- Per-title encoding optimisation with VMAF target.
- CMAF packaging with DASH support.
- GPU partitioning with MIG for A100.
- Job priority queue with fair-share scheduling.

## 0.3.0 (2026-04-20)
- NVENC hardware encoding pipeline.
- Live transcoding with sub-2s latency.
- Watchdog for automatic encoder recovery.

## 0.2.0 (2026-02-15)
- Software encoding with x264/x265.
- HLS packaging with multi-resolution ladder.

## 0.1.0 (2026-01-01)
- Basic demuxing and single-resolution encoding.
