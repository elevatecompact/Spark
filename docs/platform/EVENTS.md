# Events Platform

The events platform enables creators and the SPARK team to organize, promote, and host live and scheduled community events.

## Event Types

Live streams are real-time broadcasting events with chat, gifting, and viewer interaction. Scheduled premieres are pre-recorded videos that debut at a scheduled time with a watch party experience. Creator meetups are multi-creator events with coordinated streaming, co-streaming, and audience cross-promotion. Platform events are official SPARK events including award shows, tournaments, and holiday specials.

## Event Management

Creators can create events through the Creator Studio, specifying the event type, schedule, description, promotional artwork, and co-hosts. Events appear on the creator's channel and in the platform events calendar. RSVP functionality allows viewers to indicate attendance and receive reminders. Event pages include countdown timers, notification opt-in, and sharing options.

## Event Features

Co-streaming allows multiple creators to stream simultaneously, with their streams displayed side-by-side or in a mosaic layout. Guest invitations enable creators to invite other users as on-screen participants. Fundraising integration during events enables donation drives and charity streams. Moderation controls allow event-specific moderation rules, slow mode, and emote-only chat.

## Event Analytics

Post-event analytics provide comprehensive metrics including peak concurrent viewers, average watch time, total views, engagement metrics, gift revenue, and new follower acquisition. Comparative analytics show performance relative to previous events.

## Event Discovery

The events calendar displays upcoming events with filtering by date, category, and creator. Event recommendations surface relevant events to interested viewers. Event reminders are sent through the notification system.
