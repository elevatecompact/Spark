# translation-service — Changelog
## [1.2.0] — 2026-06-14
- Real-time chat translation via WebSocket streaming, human review queue with approval workflow, translation memory with fuzzy matching (0.8 threshold). Provider failover (DeepL→Google on 5xx).
## [1.1.0] — 2026-04-08
- Batch translation API (up to 100 texts), language detection endpoint, translation memory cache (24h TTL), usage analytics per provider.
## [1.0.0] — 2026-01-20
- Single text translation (DeepL), language detection, 40+ language support, basic caching.
