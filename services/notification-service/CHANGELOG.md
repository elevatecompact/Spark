# notification-service — Changelog
## [1.3.0] — 2026-06-16
- SMS channel (Twilio integration), digest emails (daily/weekly with cron scheduling), notification categories for grouping, rich push with images. FCM v1 API migration.
## [1.2.0] — 2026-04-12
- In-app notification center with read tracking, push token management API, delivery analytics, A/B test templates.
## [1.1.0] — 2026-02-05
- Multi-channel routing per preference, email templates with Handlebars, batch send API.
## [1.0.0] — 2025-12-01
- Push notifications (FCM), basic email (SendGrid), in-app notifications, preference CRUD.
