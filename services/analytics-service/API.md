# analytics-service — API Contract
## Dashboards: GET /v1/dashboards/creator/{id}, GET /v1/dashboards/viewer/{id}, GET /v1/dashboards/admin
## Metrics: GET /v1/metrics/realtime, GET /v1/metrics/historical, POST /v1/metrics/query (SQL-like DSL)
## Reports: POST /v1/reports/generate, GET /v1/reports/{id}, GET /v1/reports/templates
## Events: POST /v1/events/track, POST /v1/events/batch
## Funnels: POST /v1/funnels/define, GET /v1/funnels/{id}/analyze, GET /v1/funnels/{id}/results
## Export: POST /v1/export/csv, POST /v1/export/json
