# licensing-service — Changelog
## [1.2.0] — 2026-06-14
- Revenue share rate type (percentage of revenue), royalty dispute workflow, automated compliance alerts (unlicensed content detection), geo-restriction enforcement at stream/playback level, monthly royalty statements with PDF export.
## [1.1.0] — 2026-04-08
- Usage recording API with deduplication, royalty calculation batch job, royalty payout integration with wallet service, license approval workflow (draft→pending→active).
## [1.0.0] — 2026-01-20
- License CRUD, content rights registration, basic usage tracking, fixed and per-use rate types.
