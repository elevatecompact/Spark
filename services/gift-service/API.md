# gift-service — API Contract
## Catalog: GET /v1/gifts, GET /v1/gifts/{id}, POST/PATCH /v1/gifts (admin)
## Sending: POST /v1/gifts/send, POST /v1/gifts/send-batch (max 50), POST /v1/gifts/subscription
## My Gifts: GET /v1/gifts/received, GET /v1/gifts/sent, GET /v1/gifts/{id}/status
## Gift Cards: POST /v1/gift-cards/purchase, POST /v1/gift-cards/redeem, GET /v1/gift-cards/{code}
## Campaigns: POST /v1/campaigns (creator), GET /v1/campaigns, POST /v1/campaigns/{id}/match
## Analytics: GET /v1/analytics/top-gifts, GET /v1/analytics/leaderboard
