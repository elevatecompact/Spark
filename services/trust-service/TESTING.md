# trust-service — Testing Guide
## Unit: Reputation score calculation (positive vs negative weighting, decay factor), trust level threshold evaluation, risk rule evaluation engine (condition matching), signal weight assignment per category, fraud detection heuristics (rapid actions, unusual geo).
## Integration: Signal ingestion→score recalculation→trust level change event, risk assessment for high-value withdrawal, fraud case lifecycle (open→investigate→resolve), IP reputation lookup integration.
## Accuracy: Fraud detection precision/recall measured against known fraud cases. Reputation score correlation with user lifetime value.
