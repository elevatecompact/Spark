# media-service — Changelog
## [1.3.0] — 2026-06-18
- DRM support (Widevine + FairPlay) for premium content, resumable chunked uploads for large files (5GB max), CDN purge on media delete, image optimization (auto-webp, resize). GPU-accelerated transcoding with NVENC.
## [1.2.0] — 2026-04-15
- Multiple rendition profiles (720p, 1080p, source), HLS/DASH packaging, thumbnail generation at configurable intervals, S3 lifecycle policies for old media.
## [1.1.0] — 2026-02-05
- Video transcoding pipeline (FFmpeg), progressive JPEG for images, CDN integration with origin shield.
## [1.0.0] — 2025-12-01
- File upload to S3, basic image processing, CDN delivery, file metadata storage.
