# competition-service — README

## Overview
The Competition Service manages tournaments, contests, and competitive events on the Titan platform. It supports bracket-style tournaments (single/double elimination), scoring-based contests with judging panels, timed competitions with leaderboards, and prize distribution to winners. Competitions can be creator-hosted or platform-wide events.

## Purpose
Enable creators and the platform to host engaging competitive events that drive viewer participation and creator visibility. Supports multiple competition formats: elimination brackets for head-to-head matchups, score-based contests judged by panels or community voting, timed speed competitions with live leaderboards, and hybrid formats combining qualifiers and finals. Handles participant registration, match scheduling, real-time score submission, automated bracket progression, judging workflows with multiple judges, and prize pool management with automated distribution to winners.
