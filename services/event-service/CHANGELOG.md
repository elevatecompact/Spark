# event-service — Changelog
## [1.2.0] — 2026-06-15
- Recurring event series (daily/weekly/monthly), multi-session events with schedule/tracks, hybrid event support (external platform links), event reminder system (24h/1h before). Ticket inventory with optimistic locking.
## [1.1.0] — 2026-04-08
- Ticketed events with paid tiers (wallet integration), RSVP for free events, event discovery with date/category filters, post-event analytics.
## [1.0.0] — 2026-01-20
- Event CRUD, basic ticketing, calendar integration, event page with description and stream link.
