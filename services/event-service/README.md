# event-service — README

## Overview
The Event Service manages virtual and in-person events on the Titan platform. It handles event creation and promotion, ticketing and RSVP management, scheduling with calendar integration, live event coordination (stream start/end, Q&A sessions), and post-event analytics. Events can be free, ticketed, or subscription-gated.

## Purpose
Enable creators to host and monetize events beyond regular streams. Supports one-time events (concerts, workshops, meetups), recurring event series (weekly shows, classes), and multi-session conferences with tracks. Each event can have a dedicated page with description, schedule, speakers, ticketing tiers, and chat room. Events can be live-streamed within Titan or linked to external platforms for hybrid events.
