# moderation-service — Changelog
## [1.3.0] — 2026-06-16
- Real-time stream video monitoring (frame-by-frame analysis every 5s), appeal workflow with moderator review, auto-action accuracy dashboard, rule versioning with rollback. ML model accuracy improved from 89% to 94%.
## [1.2.0] — 2026-04-10
- Image moderation (NSFW, violence detection), report categorization, queue prioritization by severity, moderator performance analytics.
## [1.1.0] — 2026-02-01
- Configurable rule engine with regex and keyword patterns, human review dashboard, action reversal (appeal), batch scan API.
## [1.0.0] — 2025-12-10
- Text moderation with ML toxicity classifier, automated actions (warn/remove/suspend), content reporting API, basic review queue.
