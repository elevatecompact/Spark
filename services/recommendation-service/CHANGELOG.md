# recommendation-service — Changelog
## [1.2.0] — 2026-06-15
- Real-time feature computation from Kafka streams (sub-second freshness), trending feed with time-decay scoring, "explain why recommended" with attribution. ONNX runtime migration, inference 200ms→50ms.

- Collaborative filtering model v2, cold start for new creators, diversity sampling (avoid same-category saturation). A/B testing framework.

- Personalized home feeds, up-next suggestions, TensorFlow model training pipeline, embedding storage in PostgreSQL.
