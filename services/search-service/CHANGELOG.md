# search-service — Changelog
## [1.2.0] — 2026-06-14
- Semantic search via dense embeddings (BERT-based), personalized ranking (boost preferred categories), autocomplete with recent searches, synonym expansion with curated sets. ES cluster expanded to 3 nodes.
## [1.1.0] — 2026-04-05
- Faceted filtering (category, date, language), multi-index search across content types, search analytics logging. Typo tolerance with fuzzy matching.
## [1.0.0] — 2026-01-20
- Full-text search on creator/stream/recording indices, autocomplete with edge n-grams, basic relevance scoring, cursor pagination.
